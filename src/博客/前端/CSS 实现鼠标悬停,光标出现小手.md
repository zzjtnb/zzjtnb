用 JS 使鼠标变小手 onmouseover(鼠标越过的时候)

```JavaScript
onmouseover="this.style.cursor='hand'"
```

cursor 其他取值

```json
auto                    ：标准光标  
default                 ：标准箭头  
pointer, hand                   ：手形光标  
wait                     ：等待光标  
text                      ：I形光标  
vertical-text          ：水平I形光标  
no-drop                ：不可拖动光标  
not-allowed           ：无效光标  
help                     ：帮助光标  
all-scroll         ：三角方向标  
move                     ：移动标  
crosshair           ：十字标  
e-resize  
n-resize  
nw-resize  
w-resize  
s-resize  
se-resize  
sw-resize

```
